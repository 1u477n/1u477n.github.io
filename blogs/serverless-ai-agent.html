Serverless AI Agent Architecture
Securely Embedding Gemini into Static Websites

A comprehensive guide to embedding a smart, context-aware AI Agent into a static website without exposing API keys. This setup features local rate-limiting for cost protection, a Serverless Middleman, and a simulated typewriter effect for premium UX.

Here is the complete, professional documentation for implementing the Serverless AI Agent Architecture. You can use this guide as a reference to standardize deployments for your consulting clients, or use it to build out a custom bot — like your own "Virtual L.U.A.T" (Logical Unit for Analytical Translation) — for your personal blog.

1) Overview

This architecture allows you to embed a smart, context-aware AI Agent into a static website (like GitHub Pages) without exposing your secret API keys to the public.

It uses a Serverless Middleman (Netlify Functions) to securely route requests to the Google Gemini API, while the frontend handles advanced UX features like local rate-limiting (cost protection) and a simulated streaming (typewriter) effect.

Key Features

Security: API keys are hidden on the backend. Cross-Origin Resource Sharing (CORS) allows secure cross-domain communication.
Cost Protection: Client-side rate limiting using browser localStorage prevents API abuse.
Premium UX: Employs a dynamic glass-morphism UI, modern message bubbles, and a simulated typewriter effect for instant feedback.

2.1 Explanation of the Flow

Frontend Assembly: The browser takes the user's short question and secretly appends a massive "System Prompt" (your resume/context) so the AI knows how to act.

The Middleman: Because GitHub Pages is static, it cannot hold secret keys. It sends the payload to Netlify. Netlify acts as a secure vault, attaches your billing API key, and forwards it to Google.

Sanitization: Google sends back a massive JSON object with metadata. Netlify strips out the junk and sends only the exact text string back to the browser to keep the payload lightweight.

3) Step-by-Step Implementation Guide

Phase 1: The Backend (Netlify Function)

This step creates the secure bridge between your website and Google.

Step 1.1: Create the directory structure
In your GitHub repository, create the following exact folder structure at the root level:
netlify/functions/chat.js

Step 1.2: Add the Serverless Code
Paste the following Node.js code into chat.js. This code uses the v1 CommonJS format, which is universally compatible with all Netlify environments.

// File: netlify/functions/chat.js
exports.handler = async function(event, context) {
  // 1. Handle CORS Pre-flight (Crucial for Cross-Domain requests)
  if (event.httpMethod === "OPTIONS") {
    return {
      statusCode: 200,
      headers: {
        "Access-Control-Allow-Origin": "*", // Allows GitHub Pages to connect
        "Access-Control-Allow-Headers": "Content-Type",
        "Access-Control-Allow-Methods": "POST, OPTIONS"
      },
      body: "ok"
    };
  }

  try {
    // 2. Parse the payload from the frontend
    let body;
    try {
        body = JSON.parse(event.body);
    } catch(e) {
        body = event.body; 
    }
    
    const userMessage = body?.prompt;
    if (!userMessage) throw new Error("No prompt provided.");

    // 3. Securely retrieve the API key from environment variables
    const apiKey = process.env.GEMINI_API_KEY;
    if (!apiKey) throw new Error("GEMINI_API_KEY is missing.");

    // 4. Target the active Gemini model
    const apiUrl = `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=${apiKey}`;

    // 5. Fetch response from Google
    const response = await fetch(apiUrl, {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ contents: [{ parts: [{ text: userMessage }] }] })
    });

    if (!response.ok) {
      const errorText = await response.text();
      throw new Error(`Gemini API Error (${response.status}): ${errorText}`);
    }

    const data = await response.json();

    // 6. Safety check for unexpected Google response structures
    if (!data.candidates || !data.candidates[0] || !data.candidates[0].content) {
        throw new Error("Unexpected response structure from Google.");
    }

    const botReply = data.candidates[0].content.parts[0].text;

    // 7. Return successful sanitized string to frontend
    return {
      statusCode: 200,
      headers: {
        "Content-Type": "application/json",
        "Access-Control-Allow-Origin": "*"
      },
      body: JSON.stringify({ reply: botReply })
    };

  } catch (error) {
    // 8. Error handling
    console.error("Backend Error:", error.message);
    return {
      statusCode: 500,
      headers: {
        "Content-Type": "application/json",
        "Access-Control-Allow-Origin": "*"
      },
      body: JSON.stringify({ error: error.message })
    };
  }
};

Step 1.3: Set the Environment Variable

  • Log into your Netlify dashboard.
  • Go to Site configuration > Environment variables.
  • Add a new variable named GEMINI_API_KEY and paste your Google AI Studio key as the value.

Step 2.2: Implement the Intelligence Engine (JavaScript)
This script handles the context injection, quota limits, and the typewriter effect. Note the use of the absolute Netlify URL to ensure the fetch works across domains.

<script>
    const trigger = document.getElementById('agentTrigger');
    const windowEl = document.getElementById('agentWindow');
    const closeBtn = document.getElementById('agentClose');
    const chatArea = document.getElementById('chatArea');
    const input = document.getElementById('chatInput');
    const sendBtn = document.getElementById('sendBtn');

    // UI Toggles
    if(trigger) {
        trigger.addEventListener('click', () => { windowEl.classList.add('active'); input.focus(); });
        closeBtn.addEventListener('click', () => windowEl.classList.remove('active'));
    }

    // SYSTEM PROMPT (Hidden from User)
    // E.g., You can tell it to act as Virtual L.U.A.T to represent you.
    const systemPrompt = `You are Virtual L.U.A.T, a helpful AI assistant. 
    Keep your answers concise, strictly under 3 sentences. Be professional and direct.
    User question: `;

    async function handleMessage() {
      const text = input.value.trim();
      if(!text) return;

      // 1. COST PROTECTION: LocalStorage Daily Limit
      const limitKey = "ai_daily_usage";
      const maxQuestions = 3;
      const today = new Date().toLocaleDateString();

      let usage = JSON.parse(localStorage.getItem(limitKey)) || { count: 0, date: today };

      if (usage.date !== today) {
        usage = { count: 0, date: today }; // Reset at midnight
      }

      if (usage.count >= maxQuestions) {
        const limitMsg = document.createElement('div');
        limitMsg.className = 'msg msg-bot';
        limitMsg.innerHTML = `Daily limit reached to conserve resources. Please try again tomorrow.`;
        chatArea.appendChild(limitMsg);
        chatArea.scrollTop = chatArea.scrollHeight;
        input.value = ''; 
        return; 
      }
      
      // 2. Process User Message
      const userMsg = document.createElement('div');
      userMsg.className = 'msg msg-user';
      userMsg.textContent = text;
      chatArea.appendChild(userMsg);
      input.value = '';
      chatArea.scrollTop = chatArea.scrollHeight;

      // Increment quota and save
      usage.count++;
      localStorage.setItem(limitKey, JSON.stringify(usage));

      // 3. Display Loading Indicator
      const botMsg = document.createElement('div');
      botMsg.className = 'msg msg-bot';
      botMsg.innerHTML = "Thinking...";
      chatArea.appendChild(botMsg);
      chatArea.scrollTop = chatArea.scrollHeight;

      try {
        const fullPrompt = systemPrompt + text;

        // 4. CALL THE BACKEND (Use absolute URL if hosted separately)
        const response = await fetch('https://YOUR_NETLIFY_URL.netlify.app/.netlify/functions/chat', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ prompt: fullPrompt })
        });

        const data = await response.json().catch(() => null);

        if (!response.ok) {
            usage.count--; // Refund quota on API failure
            localStorage.setItem(limitKey, JSON.stringify(usage));
            throw new Error(data && data.error ? data.error : "HTTP Error");
        }

        if (data && data.reply) {
            const fullText = data.reply.replace(/\n/g, '<br>');
            botMsg.innerHTML = ""; 
            
            let i = 0;
            let isTag = false;
            
            // 5. TYPEWRITER EFFECT RENDERING
            function typeWriter() {
                if (i < fullText.length) {
                    let char = fullText.charAt(i);
                    if (char === '<') isTag = true;
                    
                    botMsg.innerHTML += char;
                    i++;
                    
                    if (isTag) {
                        if (char === '>') isTag = false;
                        typeWriter(); // Skip delay for HTML code rendering
                    } else {
                        chatArea.scrollTop = chatArea.scrollHeight;
                        setTimeout(typeWriter, 10); // 10ms speed per character
                    }
                }
            }
            typeWriter();

        } else {
            throw new Error("Received an empty response.");
        }

      } catch (error) {
        console.error("Error:", error);
        botMsg.innerHTML = "<span style='color: red;'><b>System Error:</b> " + error.message + "</span>";
      }
      chatArea.scrollTop = chatArea.scrollHeight;
    }

    sendBtn.addEventListener('click', handleMessage);
    input.addEventListener('keypress', (e) => { if(e.key === 'Enter') handleMessage(); });
</script>

4) Troubleshooting & Maintenance

Error: "Unable to reach the neural server" / CORS Error: Ensure your frontend fetch call is using the https://... absolute URL of your Netlify deployment, not a relative /.netlify/... path.

Error: "Gemini API Error (404)": This indicates the model name in your chat.js file is either deprecated or unsupported by your specific API key. Ensure you are using a current, active model tier like gemini-2.5-flash.

Resetting your Quota for Testing: Open your browser's Developer Tools (F12) -> Application/Storage Tab -> Local Storage -> Delete the ai_daily_usage key. Refresh the page.
